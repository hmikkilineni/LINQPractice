# LINQ Practice

C# console app demonstrating LINQ queries on integers and objects.

## What I Practiced
- .Where() - filtering
- .Select() - transforming
- .OrderBy() - sorting
- .GroupBy() - grouping
- .Count(), .First(), .Average() - aggregates
- Chaining multiple LINQ methods

## Concepts Used
- LINQ with integers
- LINQ with custom objects (Employee class)
- GroupBy with average salary and count

## Tech
- C# / .NET 8
